# Sistema de Cadastro de Clientes 

Um sistema simples e funcional desenvolvido em **Python**, permitindo cadastrar e listar clientes com validações de **CPF**, **e-mail** e **telefone**, além de armazenamento persistente em **CSV**.

Este projeto foi criado com foco em estudos, prática de backend e construção de portfólio.

---

## Funcionalidades

- Cadastro de clientes  
- Validação de CPF (11 dígitos)  
- Validação de e-mail (regex simples)  
- Validação de telefone (9–11 dígitos)  
- Listagem de todos os clientes  
- Salvamento automático em arquivo CSV  
- Menu interativo no terminal  

---

## Tecnologias Utilizadas

- **Python 3.x**
- Módulo `csv`  
- Expressões regulares (`re`)  
- Manipulação de arquivos (`open`)  
- Estruturas de dados (`listas` e `dicionários`)

---

## ▶️ Como Executar

1. Certifique-se de ter o **Python 3** instalado.  
2. Clone o repositório:
